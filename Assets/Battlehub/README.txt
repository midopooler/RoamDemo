----------------------------------------------------------------------------------
Quick Start:
----------------------------------------------------------------------------------
1. Create new scene.
2. Create transform handles using (MainMenu) Tools > Runtime Handles > Create Editor.
3. Create and select Game Object 
4. Enable object editing with Tools > Runtime Handles > Enable Editing.  
5. Click Play.

----------------------------------------------------------------------------------
Universal Render Pipeline
----------------------------------------------------------------------------------
1. Unpack UniversalRP Support [RTHandles]
2. Click (MainMenu) Tools > Runtime Handles > Use URP RenderPipelineAsset
3. Click (MainMenu) Tools > Add URP Support

----------------------------------------------------------------------------------
HD Render Pipeline
----------------------------------------------------------------------------------
1. Unpack HDRP Support [RTHandles]
2. Click (MainMenu) Tools > Runtime Handles > Use HD RenderPipelineAsset
3. Click (MainMenu) Tools > Runtime Handles > Add HDPR Support
